## 🚀 Como executar o projeto

#### Clonar o repositório

`git clone https://github.com/Sandoque/app-tictactoe.git`

#### Entrar no diretório

`cd repo-tictactoe`

#### Instalar as dependencias

`npm install`

#### Rodar o servidor

- `ng serve --open`
- Navegar em `http://localhost:4200/`
